# Major limitation

![Public market](/_images/public_market.png)

* Rangefinder only works on public Forge modules (currently).
* Cannot identify how code is used by profiles or private modules.

~~~SECTION:notes~~~

* Right now, Rangefinder can only identify dependencies from public Forge modules.
* To truly know the spread and the impact of breaking changes we need production
  usage patterns from infrastructures actually *using* modules and components.
